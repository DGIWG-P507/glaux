DGIWG Project 07 (SWE)

DGIWG GLAUX

Operationalizing NATO STANAG 4789 & OGC API - Connected Systems for Joint ISR Operations

Vision & Mandate

Conceived during the DGIWG Spring 2026 sessions in Athens, Greece, Glaux delivers an open-source suite of reference implementations and ecosystem software. Its primary mission is to provide the resources, evaluation tooling, and implementation guidance necessary to accelerate plug-and-play sensor interoperability across the Alliance.

The Glaux Software Suite

Glaux is delivered as a modular suite of tightly decoupled components designed for rapid coalition deployment:

🛡️ Glaux Server

The foundational, core OGC API - Connected Systems reference implementation handling secure compliant routing.

View Repository

📊 Glaux Web App

The operational front-end dashboard providing real-time visualization of connected platforms and sensor states.

View Repository

📡 Glaux Simulator

A robust telemetry generator simulating STANAG 4789 data streams for compliance evaluation and stress testing.

View Repository

🔄 Glaux Publisher

The ingestion pipeline serving as a bridge to translate legacy sensor networks into modern interoperable feeds.

View Repository
